关键设计决策

  • 什么时候才显示 CTA？ 只有 ① 用户消息带图 ② 模型在回复里返回 questionDetected: true 时才显示。纯文字闲聊不显示，避免打扰。
  • 为什么默认勾上"加入错题本"？ 陪学场景下用户来问 AI，大概率是不会做才问。给一个 checkbox 让他能反勾，比每次问"加哪里"省一步。
  • 点"不用了" 会怎样？ CTA 行直接消失（前端 state 标记），不写库。AI 不会再追问。
  • 已入库后能否撤销？ 不能直接撤。题已经在题库列表里，用户可以去题库管理删（已有功能）。
  • 技术实现要点： chatStudy 需要让模型在回复里附带结构化字段（如 <extracted-question>{"stem":...}</extracted-question>），前端解析后渲染 CTA + 携带这份 JSON 跳确认页。图片要随之传过去（dataUrl 暂存 sessionStorage）。
